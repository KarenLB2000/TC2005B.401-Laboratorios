¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Dado que nunce he programado en Java, tomaré como referencia al artículo de ISDI Coders (2017) respecto a este tema. Así pues, se presenta en el artículo que algunas de las diferencias enter ambos lenguajes es:

  • Java crea aplicaciones que tras ser compiladasse pueden ejecutar en navegadores a través de una máquina virtual, mientras que Javascript se ejecuta nativamente en los navegadores sin necesidad de ninguna máquina virtual.
  • Java es un lenguaje fuertemente tipado y Javascript es débilmente tipado.
  • Java es un lenguaje de programación orientado a objetos puramente, y Javascript es un lenguaje basado en prototipos.

En tanto a las semejanzas, aunque algo obvio, éstos tienen nombres parecidos.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

La clase Date permite trabajar con fechas y horas en JavaScript. Algunos de los métodos que de esta clase incluyen (MDN Web Docs, 2022):

  • Date.now(): Devuelve el valor numérico correspondiente a la hora actual.
  • Date.parse(): Transforma una cadena que representa una fecha, y devuelve el número de milisegundos desde el 1 de enero de 1970, hora local 00:00:00.
  • Date.toUTCString(): Convierte una fecha en una cadena, utilizando la zona horaria UTC.
  • Date.setTime(): Establece el objeto Date en la hora representada por una cantidad de milisegundos desde el 1 de enero de 1970, 00:00:00 UTC.
  • Date.getTimezoneOffset(): Devuelve la diferencia, en minutos, entre una fecha evaluada en la zona horaria UTC y la misma fecha evaluada en la zona horaria local.

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

En tanto a los métodos del objeto Array, algunos de ellos son (MDN Web Docs, 2022):

  • Array.find(): Devuelve el valor del primer elemento del arreglo que cumple la función de prueba proporcionada.
  • Array.sort(): Ordena los elementos de un arreglo localmente y devuelve el arreglo ordenado
  • Array.push(): Añade uno o más elementos al final de un arreglo y devuelve la nueva longitud del arreglo.
  • Array.reverse(): Invierte el orden de los elementos de un arreglo
  • Array.join(): Une todos los elementos de una matriz en una cadena y devuelve esta cadena.

¿Cómo se declara una variable con alcance local dentro de una función?

El alcance de la variable generalmente se define por el lugar de su declaración. Una variable declarada dentro de una función tiene alcance local y, por lo tanto, se denomina variable local. Las variables globales se definen fuera de las funciones y tienen acceso a todas las funciones y no solo a una.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Si bien las variables globales pueden utilizarse para compartir variables entre funciones de forma sencilla, en la práctica su uso conlleva diversos efectos colaterales como lo puede ser la alteración no deseada del contenido de una variable global dentro de una función, lo cuál generaría errores cuya correción puede ser muy ardua (impacto del cambio). Además, entre tales otros ejemplos de efectos colaterales, tenemos también a las depencias ocultas y el orden de destrucción (Stack Overflow, 2016).
