¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Si bien las variables globales pueden utilizarse para compartir variables entre funciones de forma sencilla, en la práctica su uso conlleva diversos efectos colaterales como lo puede ser la alteración no deseada del contenido de una variable global dentro de una función, lo cuál generaría errores cuya correción puede ser muy ardua (impacto del cambio). Además, entre tales otros ejemplos de efectos colaterales, tenemos también a las depencias ocultas y el orden de destrucción (Stack Overflow, 2016).
